Quiz
idea: quiz About Christmas; whenever you get something Right you get a score. when you get something wrong it adds to another score. in the end, depending on which score is higher, it says you are a good kid or naughty. 

how: Radio Surveys, score: if true scoreTrue = scoreTrue + 1 (same for false) 
if value scoreTrue >= value scoreFalse good kid text
if value scoreTrue < value scoreFalse naughty kid text
